# Euclid Circular B Font Files

## Instructions

Place your Euclid Circular B font files in this directory with the following names:

- `EuclidCircularB-Light.woff2` (and .woff)
- `EuclidCircularB-Regular.woff2` (and .woff)
- `EuclidCircularB-Medium.woff2` (and .woff)
- `EuclidCircularB-SemiBold.woff2` (and .woff)
- `EuclidCircularB-Bold.woff2` (and .woff)

## Where to get the font

1. If you have the font files from your design team, copy them here
2. Ensure you have the proper license to use the font
3. Convert TTF/OTF files to WOFF2/WOFF format if needed using online converters

## Font weights mapping

- Light: 300
- Regular: 400
- Medium: 500
- SemiBold: 600
- Bold: 700

The font is already configured in `src/index.css` and will load automatically once the files are placed here.
